Welcome to everpeace's personal maven repository
What is?

This is a maven artifacts repository of everpeace. Source code of my artifacts is hosted mainly from my personal github repository(https://github.com/everpeace/everpeace-personal).

How to Use?
  1. To browse my artifact, see https://github.com/everpeace/maven-repository
  2. To colaborate my artifacts, add repository entries in your pom such like: 
    <project>
  ...
  <repositories>
    ...
    <repository>
      <id>org.everpeace.repository</id>
      <url>http://everpeace.github.com/maven-repository/release</url>
    </repository>
    <repository>
      <id>org.everpeace.repository.snapshot</id>
      <url>http://everpeace.github.com/maven-repository/snapshot</url>
    </repository>
  </repositories>
  ...
</project>
